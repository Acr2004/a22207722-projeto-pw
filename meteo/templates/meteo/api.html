API de Previsão do Tempo

A API de Previsão do Tempo fornece informações meteorológicas para várias cidades. Abaixo estão os detalhes dos endpoints disponíveis, incluindo parametros, métodos de requisição, e exemplos de respostas.

  • 1. Listar Cidades
    Endpoint: /api/cidades/
    Método: GET
    Descrição: Retorna uma lista de todas as cidades disponíveis.
    Parâmetros:Nenhum.
    Resposta Exemplo:

    { "cidades": [ "Lisboa", "Porto", "Braga", "Aveiro" ] }

  • 2. Previsão Meteorológica de Hoje
    Endpoint: /api/weather/today/
    Método: GET
    Descrição: Retorna a previsão meteorológica para hoje para uma cidade específica.
    Parâmetros:
    city_id (obrigatório): O ID da cidade para a qual deseja a previsão do tempo.

    { { "owner": "IPMA", "country": "PT", "data": [ { "precipitaProb": "91.0", "tMin": "12.3", "tMax": "19.7", "predWindDir": "NE", "idWeatherType": 9, "classWindSpeed": 3, "longitude": "-8.1236", "forecastDate": "2023-12-16", "classPrecInt": 2, "latitude": "36.847" }, "globalIdLocal": 1110600, "dataUpdate": "2023-12-16T12:56:12" } }
